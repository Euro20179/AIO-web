Calculator Script

Calculator script is a simple scripting language with an incomplete api that was created for the calculator mode.

Due to the lackluster api, I recommend just using JS where possible.

This language has a couple of unique features. Every statement is an expression that returns a value.

There is a list version of while loops, and for(each) loops, that instead of just returning the last value, put the last value of each iteration into a list, and that is what is returned

Calling a number is the same as multiplication

Most functions try to type juggle into the correct type. Exceptions will be mentioned explicitely

Unorthadox Syntax

expr (=> expr)+: map all items from the lefthand side expr through each arrow one at a time. Each expr after an arrow is treated as a closure. To access the current item, use arg0.

expr (?> expr)+: similar to the map arrow, but is a filter arrow.

For Lists. A list created by a for loop

Builtin Functions

Note: booleans do not exist, instead it is either 1 (true) or 0 (false)

Grammar

NUMBER       :    [0-9]+ | [0-9]*.[0-9]+
STRING       : '"' ... '"'
WORD         :    [A-Za-z][A-Za-z0-9]*
atom         :    NUMBER
                    | STRING
                    | "(" program ")"
                    | var-def
                    | func-def
                    | while-loop
                    | while-list
                    | for-loop
                    | for-list
                    | foreach-loop
                    | foreach-list
                    | if-statement
func-call    :    WORD "(" (expr ",")* ")"
prop-access  :   "[" expr "]"
left-un-op   :   ("+" | "-")? atom
right-un-op  :   left-un-op (func-call | prop-access)?
factor       :   right-un-op (("*" | "/") right-un-op)*
term         :   factor (("+" | "-") factor)*
comp         :   term ((">" | "<" | "==" | "<=" | ">=") comp)*
logical-comp :   comp (("and" | "or" | "xor" | "nand") logical-comp)*
pipe-ops     :   logical-comp (("=>" | "?>") logical-comp)*
pipe         :   pipe-ops ("-> pipe-ops)*

func-def     :  "var()" WORD "=" program "rav"
var-def      :  (("var" | "let") WORD "=" expr) | func-def

while-loop   : "while" expr "do" expr ("fi" | "elihw" | "done")
while-list   : "while" expr ":" expr ("fi" | "elihw" | "done")

foreach-loop : "foreach" WORD "in" expr "do" expr "rof"
foreach-list : "foreach" WORD "in" expr ":" expr "rof"

for-loop     : "for" WORD "=" expr "," expr "do" expr "rof"
for-list     : "for" WORD "=" expr "," expr ":" expr "rof"

if-statement : "if" expr "do" atom ("else" atom)? "fi"?

expr         :  pipe

program: (expr ";")*
