# AIO-WEB

The web frontend for the [AIO LIMAS](https://github.com/euro20179/aio-limas).

# Setup

See the sample `server-config-example.ini` file, and edit as needed.
Once finished rename it to `server-config.ini` and move to the `www` directory

# Running

1. `./build`
2.
    * `./quickstart.sh`
    * OR see `./example.nginx` for an nginx installation
